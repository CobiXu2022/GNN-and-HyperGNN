# GNN and HyperGNN Projects Collection

A collection of four Graph Neural Network and Hypergraph Neural Network projects.

## Projects

- [AML](./AML/) - Anti-money laundering detection (GAT model)
- [aml-elliptic-gnn](./aml-elliptic-gnn/) - AML detection on Elliptic dataset (multiple GNN models comparison)
- [HyperGCN](./HyperGCN/) - Hypergraph Convolutional Networks
- [UniGNN](./UniGNN/) - Unified Graph and Hypergraph Neural Networks framework

## Installation

```bash
pip install torch torch-geometric numpy pandas scikit-learn matplotlib
```

For detailed usage instructions, please refer to each project's README.
